# **App Name**: Digital Canvas

## Core Features:

- Project Showcase: Display a portfolio of projects with titles, descriptions, and technologies used.
- Category Filters: Filter projects by category (e.g., Marketing, Data Science, Development).
- Contact Form: Allow visitors to send messages with name, email, subject, and message fields; the website also prompts users for POPIA consent.
- Dynamic Recommendations: AI tool to suggest relevant projects based on the user's expressed interests in the contact form's message.
- Experience Timeline: Showcase professional experience with job titles, descriptions, and dates.
- Skill Visualization: Generative AI to generate a description for all skills listed in the resume in bullet point form.

## Style Guidelines:

- Primary color: Vibrant Yellow (#E4D00A) for attracting attention and indicating specialized service.
- Background color: Light gray (#EAE9E5) providing a subtle, clean backdrop.
- Accent color: Deep blue (#3777FF) for providing high contrast while remaining analogous to the yellow of the existing page.
- Body and headline font: 'Inter', sans-serif, for its modern and neutral appearance.
- Simple line icons for services and contact information.
- Clean, modern layout with clear sections for easy navigation.
- Subtle hover effects on portfolio items and buttons to add interactivity.
